## 🚀 Sobre mim

Atuo no mercado financeiro construindo plataformas de dados de ponta a ponta — da modelagem no data warehouse até a entrega do indicador para o negócio. Na **Lotus Capital**, trabalho sobre o Microsoft Fabric em arquitetura medallion, desenvolvendo pipelines de ETL, modelagem dimensional, dashboards estratégicos em Power BI (DAX) e uma aplicação de analytics própria em Python (Flask) conectada ao Fabric via SQL.

Traduzo regras de negócio de gestão de patrimônio (AuC, NNM, captação, churn, NPS, ativação) em métricas confiáveis, atuando na interseção entre dados, tecnologia e negócio. Formado em Análise e Desenvolvimento de Sistemas pela PUC Minas.
